# Thermal Monitoring System Documentation

## Overview
The Thermal Monitoring System is designed to monitor and analyze temperature data in real-time. It is ideal for applications where temperature fluctuations may impact performance, safety, or efficiency.

## Features
- Real-time temperature monitoring
- Data logging for temperature analytics
- Alerts for temperature thresholds
- User-friendly interface for data visualization

## System Architecture
The system is built on a modular architecture comprising:
- Sensor Module: Captures temperature data using sensors.
- Data Transmission Module: Sends data to a server for processing.
- User Interface: Displays data graphs and alerts.

## Installation
1. Clone this repository:
   
   ```bash
   git clone https://github.com/nadal243/PROJET---Programmation-informatique.git
   ```
2. Navigate to the thermal_monitoring_system directory:
   
   ```bash
   cd PROJET---Programmation-informatique/thermal_monitoring_system
   ```
3. Install required dependencies:
   
   ```bash
   npm install
   ```
4. Start the system:
   
   ```bash
   npm start
   ```

## Usage
- Access the user interface via the provided local URL.
- Monitor real-time temperature data and configure alerts.
